HTML Summary

Tag Purpose Example Explanation
'<html>' Defines the start and end of an HTML document. <html>....</html> The root element that encloses all other HTML elements.
'<head>' Contains meta-information about the document. <head>....</head> Holds information such as the title, character set, and linked resources.
'<title>' Sets the title of the document. <title> My Page </title> The text displayed on the browser tab or window title bar.
'<body>' Contains the content of the document. <body> .... </body> Encloses all the visible content of the webpage, such as text and images.
'<h1>' to '<h6>' Define headings, with '<h1>' being the highest level. <h1> Heading </h1> Used for headings, where '<h1>' is the largest and '<h6>' is the smallest.
'<p>' Defines a paragraph. <p> Paragraph </p> Represents a block of text with space above and below.
'<a>' Creates a hyperlink. <a href="https://example.com">Link </a> Links to another webpage or resource.
'<img>' Embeds an image. < img src="image.jpg" alt="Description" > Displays an image, with attributes for source and alternative text.
'<ul>' Creates an unordered (bulleted) list. <ul><li> item1 </li></ul> Represents a list with bullet points.
'<ul>' Creates an unordered (bulleted) list. <ul><li> item1 </li></ul> Represents a list with bullet points.
